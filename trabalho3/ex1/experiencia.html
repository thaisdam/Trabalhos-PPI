Descrição dos Projetos

  • Sistema de Gestão de Benefícios:

    Desenvolveu uma aplicação web para gerenciamento de benefícios corporativos, permitindo cadastro e controle de benefícios concedidos a funcionários. Utilizou Django no backend, React.js no frontend e PostgreSQL para armazenamento dos dados. Implementou autenticação com JWT e um painel administrativo para visualização de relatórios.

  • Plataforma de Biblioteca Digital

    Criou um sistema de biblioteca digital com funcionalidades de empréstimo, devolução e reserva de livros. Desenvolvido com Flask, Bootstrap e MySQL, integrando um sistema de notificações via e-mail. Implementou pesquisa otimizada utilizando indexação de dados e consultas SQL avançadas.

  • Chatbot de Atendimento para Empresas:

    Desenvolveu um chatbot baseado em inteligência artificial para responder dúvidas frequentes de clientes em um site corporativo. Utilizou Python com NLTK para processamento de linguagem natural e integrou a solução ao Telegram e WhatsApp.

  • Dashboard de Análise de Vendas:

    Criou um painel interativo para análise de vendas, permitindo a visualização de métricas e gráficos dinâmicos. Utilizou Pandas e Matplotlib para análise de dados e Dash (Plotly) para a interface interativa. Desenvolveu filtros personalizados para análises específicas por período e categoria de produto.
